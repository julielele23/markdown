| **Element**     |  **Markdown syntax**               |
| ---------- | ------------------------------ |
|**Heading** | # level1 |
|        | ## level2 |
|        | ### level3|

| | ========== |
|   | heading level 2 |
| | ----------------- |

| | 3. third item |
| **Unorderd list** | - first item |

| | - third item |

|**Horizontal rule** | --- |
